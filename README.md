# food-pos

Bespoke fastfood/bakery operating framework focused on sales capture, inventory control, cost discipline, and profitability analytics.

## Included in this repo
- `BUSINESS_INTELLIGENCE_FRAMEWORK.md`: End-to-end framework for building and running a data-intensive food business software stack.
- `templates/daily_sales_cash_log_template.csv`: Daily closure template for sales channels, expenses, cash checks, and owner sign-off.
- `templates/weekly_cost_status_template.csv`: Weekly performance/cost control template with issue and action tracking.
- `templates/monthly_survival_dashboard_template.csv`: Monthly survival and runway template for operating decisions.
- `templates/monthly_unit_economics_template.csv`: Unit economics template for margin and break-even analysis.

## Next step
Use the framework and templates as your requirements and reporting baseline while implementing the full POS + inventory + production + finance platform.
